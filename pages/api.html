API

Want to use Vestigo data inside your own application? The API below documents how to store and retrieve information from Vestigo. If you have questions or have issues with the API, contact support@novaslp.net

Item types

Currently vestigo supports the following item types: music - CD, Records, Tapes
movie- DVD, Blu-Ray, HD-DVD, VCD, VHS
toy - Matchbox, Building sets, Dolls
book - Novels, text books
boardgame - Board games
digitalgame - PC Game, Videogames
tool - Powertools, hand tools

Item Attributes

All items stored in Vestigo can have the following attributes stored for them.
type - Set by the save attribute
title - The title of the item
text - A descriptive text about the item.
photo* - A list of photos of the item
model_number - The model number of the item
url - A link to a related web page
qty - Quantity of the item
date_purchase - The date the item was purchased
date_create* - Set automatically when the item is first saved
date_update* - Set automatically whne the item is updated
* - Not updatable via the API.

Item Music Attributes

Attributes specifically related to a music item.
media - The type of media the music is stored on
artist - The musician or group that performed the music

Item Movie Attributes

Attributes specifically related to a movie item.
media - The type of media the music is stored on
format - Widescreen, Full Frame, HD resolution
language - Native language of movie
producer - Producer of movie

Item Toy Attributes

Attributes specifically related to a toy item.
pieces - How many pieces the toy has

Item Book Attributes

Attributes specifically related to a book item.
author - The author of the book
publisher - The publisher of the book

Item Boardgame Attributes

Attributes specifically related to a boardgame.
year_release - The release year of the board game
edition - Store information about a specific edition of a game

Item Digital Game Attributes

Attributes specifically related to a digital game.
media - The media (CD, GDROM, Cartridge) that a game is stored on.
digital_system - The system the item is designed for

Item Tool Attributes

Attributes specifically related to a tool.
manufacturer - The manufacturer of an item.
year_warranty - The length of warranty on the tool

URL

The api is available at http://vestigoapp.appspot.com/api/. At present call rates to the API are not limited. Under high load, call rates to the API from 3rd party applications are limited to maintain the website. URLs can be formed by appending the function in lower case to the base url.

list

The list function will list all items a user has in their collection with all base attributes.
XML Structure: [items]->[item]->attributes

list/*

Ex: list/music
List a specific type of item with the attributes that pertain to it.
XML Structure: [items]->[item]->attributes

save/*

Ex: save/tool
Post attributes related to this item to save the item to vestigo.
Optional Parameters: item_id - Post a numberic item_id to update an existing item.

get/_types_

Get a list of all possible attribute types that Vestigo stores predefined values for.
XML Structure: [item_types]->[item_type]->[title]

get/*

Get a list of possible values for each attribute type. Users are not limited to items from these list. The service will accept any value for each option but selecting one of these ensures data stays sorted for each user.
XML Structure: [nodes]->[node]->attributes
Optional Parameters: node_like - return 10 similar possible values
